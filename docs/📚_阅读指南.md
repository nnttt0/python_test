## 🎯 核心文件说明

### 1. 数据库层 (`database/`)

| 文件 | 作用 | 关键内容 |
|------|------|---------|
| `config.py` | 数据库配置 | MySQL连接参数、连接池设置 |
| `db.py` | 连接管理 | engine创建、Session工厂、init_db() |
| `models.py` | ORM模型 | PredictionRecord类（18个字段） |
| `services.py` | CRUD服务 | PredictionService类（5个方法） |
| `__init__.py` | 模块导出 | 导出get_db、init_db、PredictionService |

### 2. 前端层 (`templates/` + `static/`)

| 文件 | 作用 | 关键功能 |
|------|------|---------|
| `index.html` | 主页面 | 表单、上传区、图表容器、聚类区域 |
| `main.js` | 前端逻辑 | AJAX请求、图表切换、CSV导出 |
| `style.css` | 样式美化 | CSS3动画、响应式布局 |

### 3. 工具脚本

| 文件 | 作用 | 使用方法 |
|------|------|---------|
| `init_db.py` | 数据库初始化 | `python init_db.py` → 输入yes确认 |

```bash
# 输入 yes 确认删除并重建
```

### 启动完整应用
```bash
# 从项目根目录
cd person2_backend
python app.py
# 访问 http://localhost:8888
```

### 测试数据库连接
```python

db = next(get_db())
stats = PredictionService.get_statistics(db)
print(stats)
```
